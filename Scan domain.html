Scan Domaine est un script permettant de récupérer, à partir d'un nom de domaine, la list des sous-domaine assosier dans un fichier txt.
  • Langage : Python
  • IDE : Visual Studio Code

Installation

                
                    python -m pip install requests
Ou si vous êtes sous python 3 
                
                    python3 -m pip install requests

Premiere fonction : get_sub_domain

A partir d'un fichier github récupere la liste de sous-domaines avec request.get(take the informations via the link) 
                
            def get_sub_domain(tab):
                try:
                    url='https://raw.githubusercontent.com/rbsec/dnscan/master/subdomains-10000.txt'
                    req=requests.get(url, allow_redirects=True)
                except:
                    print("error")
puis récupérer chaque domaine ligne par ligne(req.text.splitLines) dans une list(listName.append) 
                
                i=0
                while i < 9985: 
                    tab.append(req.text.splitlines()[i]) 
                    i+=1
                return (tab)
